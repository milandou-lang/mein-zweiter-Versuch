# RĂ©viser les pronoms franĂ§ais

Dies ist eine statische Homepage fuer den Franzoesischunterricht zum Repetieren der Pronomen. Sie kann kostenlos mit GitHub Pages veroeffentlicht werden.

## Dateien

- `index.html`: Inhalt und Struktur der Seite auf Franzoesisch
- `styles.css`: Gestaltung in Braun- und Rottoenen, Layout und mobile Ansicht

## Auf GitHub starten

1. Erstelle auf GitHub ein neues Repository, zum Beispiel `pronoms-francais`.
2. Lade `index.html`, `styles.css` und diese `README.md` in das Repository hoch.
3. Oeffne im Repository `Settings` > `Pages`.
4. Waehle bei `Build and deployment` die Quelle `Deploy from a branch`.
5. Waehle Branch `main` und Ordner `/root`.
6. Speichere. Nach kurzer Zeit zeigt GitHub dir die Webseiten-Adresse an.

## Anpassen

- Passe die Pronomen-Beispiele an deine Klasse an.
- Ergaenze Links zu Arbeitsblaettern, Quiz oder Lernplattformen.
- Wenn du ein eigenes Bild nutzen willst, ersetze die Bild-URL in `styles.css` bei `.hero`.
